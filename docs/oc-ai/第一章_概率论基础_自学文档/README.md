# 第一章 概率论基础自学文档

这是一份“学习版”整理，不追求和原讲义一一对应，而是按自学更顺的路线组织：先建立概率直觉，再学习计算规则，然后进入常见分布、高斯模型、贝叶斯估计、指数族、马尔科夫链和 KL 散度。

## 建议阅读顺序

1. `00_学习路线.md`：先看整体地图，知道每一部分为什么要学。
2. `01_概率语言与随机变量.md`：把概率的基本语言学稳。
3. `02_条件概率_贝叶斯_MMSE.md`：掌握“看到数据后如何更新信念”。
4. `03_常见分布与高斯模型.md`：认识机器学习常用分布，重点是高斯。
5. `04_线性高斯系统与指数族.md`：理解为什么某些模型特别容易计算。
6. `05_马尔科夫链与散度度量.md`：学习序列建模和分布差异度量。
7. `06_公式速查.md`：复习前闭卷默写，复习后查漏补缺。
8. `07_自测题与练习安排.md`：用题目检测是否真的会用。
9. `08_补充例子与常见误区.md`：配套小例子、易错点和学习建议。

## 学习目标

学完后你应该能回答三类问题：

- 建模问题：这个随机现象适合用什么分布描述？
- 推断问题：看到观测数据后，未知量的概率如何更新？
- 比较问题：两个概率分布差多少，模型拟合数据好不好？

## 本章主线

> 概率论 = 描述不确定性 + 更新不确定性 + 建模数据 + 比较分布。

把它放进机器学习语境中：

- 数据有噪声，所以需要概率模型。
- 参数未知，所以需要估计或贝叶斯推断。
- 预测有不确定性，所以需要后验分布或置信度。
- 模型不完美，所以需要 KL 散度、似然等指标衡量拟合。
