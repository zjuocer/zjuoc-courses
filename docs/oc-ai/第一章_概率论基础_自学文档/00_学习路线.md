# 00 学习路线

本章建议按下面顺序学习：

1. `01_概率语言与随机变量.md`
2. `02_条件概率_贝叶斯_MMSE.md`
3. `03_常见分布与高斯模型.md`
4. `04_线性高斯系统与指数族.md`
5. `05_马尔科夫链与散度度量.md`

学习时抓住一条主线：

> 概率论用来描述不确定性；贝叶斯规则用来更新不确定性；分布和模型用来刻画数据；KL 散度用来比较模型和数据之间的差异。

优先掌握：

- 条件概率、全概率公式、贝叶斯定理。
- PMF、PDF、CDF 的区别。
- 伯努利、二项、泊松、高斯等常见分布。
- 多元高斯条件分布。
- KL 散度与最大似然估计的关系。

每学完一节，问自己三个问题：

1. 这个知识点解决什么问题？
2. 核心公式是什么？
3. 能不能举一个小例子算出来？
